Raquel's Journey to The Sun
============================

Jogo de plataforma onde Raquel embarca numa jornada para encontrar o sol na escuridão.

CONTROLES:
- Seta Esquerda: Mover para a esquerda
- Seta Direita: Mover para a direita
- Espaço: Saltar

COMO JOGAR:
1. Execute o arquivo "RaquelBeta.exe"
2. Use as setas para mover e espaço para saltar
3. Colete chaves e gemas para progredir
4. Complete todos os níveis para encontrar o sol!

Divirta-se!
